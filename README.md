# Finance Control - COMPLETO 🚀

App de controle financeiro com todas as funcionalidades.

## Recursos

✅ **Resumo do Mês** - Visualize toda sua receita, despesas e aportes
✅ **Fundo de Emergência** - Acompanhe o progresso rumo aos R$ 6.000
✅ **Investimentos em FIIs** - Registre aportes e veja a projeção de aposentadoria
✅ **Meta de Aposentadoria** - Saiba quando você terá R$ 6.000/mês passivo
✅ **Contas a Pagar** - Gerencie suas despesas mensais
✅ **Lista de Desejos** - Organize suas compras com prioridades
✅ **Configurações** - Personalize suas metas e salário
✅ **Sincronização** - Tudo salvo na nuvem (Supabase)

## Como Usar

1. Descompacte esta pasta
2. Abra Terminal/CMD
3. Execute:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/seu-usuario/finance-control.git
git push -u origin main
```

4. Aguarde 2-3 minutos
5. Clique em "Visit" no Vercel
6. **Pronto! App completo rodando!** 🎉

## Funcionalidades

### 📊 Resumo
- Receita total (salário + comissão)
- Despesas e aportes
- Sobra disponível
- Meta de aposentadoria e progresso

### 🏦 Fundo de Emergência
- Meta de R$ 6.000
- Aporte mínimo sugerido: R$ 300/mês
- Progresso visual
- Rendimento estimado CDI

### 📈 Investimentos
- Aportes em FIIs
- Projeção de renda passiva
- Recomendações de FIIs (MXRF11, HGLG11, etc)
- Cálculo automático de quanto é necessário

### 📋 Contas
- Adicione suas despesas mensais
- Acompanhe valores
- Exclua quando necessário

### 🎁 Desejos
- Organize suas compras
- Priorize por urgência
- Acompanhe o progresso

### ⚙️ Configurações
- Personalize salário base
- Ajuste metas
- Configure comissão estimada

## Dicas

💡 **Reinvestir dividendos** - Deixe os dividendos dos FIIs serem reinvestidos para acelerar o "efeito bola de neve"

💡 **Aporte extras** - Sempre que sobrar dinheiro, adicione como aporte extra em FIIs ou emergência

💡 **Revisar mensalmente** - No início de cada mês, atualize a comissão e veja o progresso

## Suporte

Seus dados estão seguros no Supabase com criptografia e autenticação.
Acesse de PC, celular ou tablet - tudo sincronizado!
